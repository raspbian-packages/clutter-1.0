4. Where to get Clutter

You can obtain the Clutter source code ready for compilation from http://www.clutter-project.org.

Clutter is also available on all major GNU/Linux distributions, in various package formats.

On OSX, Clutter is available with both Fink and MacPorts.

Binaries for Microsoft Windows are also available.
